8.9. Full dislocation loop arrayÔÉÅ

To create an array of full dislocation loops in a BCC single crystal sample,

Step 1:

Use the tool in subdirectory tools/Bcc to creat a BCC single crystal sample: bcc_reg.lmp, use the shell script in subdirectory tools/para_file_loop_array to create the parameter file: para_file;

Step 2:

Open terminal, run the following command:

1CryDisGen para_file bcc_reg.lmp
