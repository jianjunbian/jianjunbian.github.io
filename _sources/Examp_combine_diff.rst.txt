Combination of dislocations
=======================================


.. |combined|  image:: img/combined_diff.png
   :width: 40% 
   :align: bottom 

|combined|

--------

To create a combination of different types of dislocations in FCC single crystal sample, 

**Step 1:**

  Use the tool in subdirectory :doc:`tools/Fcc <Tools>` to creat an FCC single crystal sample: ``fcc_reg.lmp``;

**Step 2:**

  Open terminal, run the following command:

.. code-block:: Shell 
   :linenos:

   CryDisGen para_file fcc_reg.lmp

--------

.. admonition::  Parameter file 

   Content of the :doc:`para_file <ParaFile>`:

.. code-block:: CryDisGen
   :linenos:

    #those started with '#' are comment lines
    #crystal region
    Region
    -x      1  0  0      # lattice orientation in x-direction
    -y      0  1  0      # lattice orientation in y-direction
    -z      0  0  1      # lattice orientation in z-direction
    -lat    3.615        # lattice constant
    -pbc    1  1  1      # boundary condition flag: '1': pbc, '0': non-pbc 

    #dislcoation
    nLoop   4            # number of dislocaito loops
    Loop                 # label of a dislocation section
    -bv     1  1  1      # orientation of Burger's vector
    -bs     1/3          # scalar coefficient of Burger's vector, e.g., 1/2 in: 1/2<110>
    -dnorm  1  1  1      # normal direction of plane with dislocation loop
    -dc     0. 0  0      # center of dislocation loop
    -dr     25           # radius of dislocaiton loopS
    -lx    -1  1  0      # local x axis on slip plane 
    -ly    -1 -1  2      # local y axis on slip plane 
    -ndisl  3            # segments of a discretized dislocation loop

    Loop
    -bv    -2 -1 -1
    -bs     1/6
    -dnorm -1  1  1 
    -dc     30 0  0
    -dr     15 
    -lx    -1 -1  0
    -ly     1 -1  2
    -ndisl  3
     
    nTetrahedron 1       # number of stacking fault tetrahedron
    Tetrahedron          # label of tetrahedron
    -tc     20 20 20.5   # center of the bottom face of tetrahedron
    -tr     25           # radius of the circumscribe circle of the bottom face
    -tnorm -1 -1 -1      # outward nomral direction bottom face of tetrahedron
    -lxt    1 -1  0      # local x axis on the bottom face
    -lyt   -1 -1  2      # local y axis on the bottom face

    Loop
    -bv    -1 -1  0
    -bs     1/2
    -dnorm -1  1  1 
    -dc    30  30 -20
    -dr    15 
    -lx    -1 -1  0
    -ly     1 -1  2
    -ndisl 15

    Loop
    -bv     1  1  0
    -bs     1/2
    -dnorm  1  1  0 
    -dc     -20 -20 -0
    -dr     30 
    -lx     -1  1 -1
    -ly     -1  1  2
    -ndisl  4
