Shockley partial loop
=================================

.. |shockley|  image:: img/shockley.png
   :width: 40%
   :align: bottom 

|shockley|

--------

To create a Shockley partial dislocation loop, 

**Step 1:**

  Use the tool in subdirectory :doc:`tools/Fcc <Tools>` to creat an FCC single crystal sample: ``fcc_reg.lmp``;

**Step 2:**

  Open terminal, run the following command:

.. code-block:: Shell 
   :linenos:

   CryDisGen para_file fcc_reg.lmp

--------

.. admonition:: Parameter file 

   Content of the :doc:`para_file <ParaFile>`:

.. code-block:: CryDisGen
   :linenos:

   #those starting with '#' are comment lines
   #crystal region
   Region
   -x       1  0  0      # lattice orientation in x-direction
   -y       0  1  0      # lattice orientation in y-direction
   -z       0  0  1      # lattice orientation in z-direction
   -lat     3.615        # lattice constant
   -pbc     1  1  1      # boundary condition flag along x/y/z-axis: '1': pbc, '0': non-pbc 

   #dislcoation
   nLoop    1            # number of dislocaito loop
   Loop                  # label of a loop section
   -bv     -1  2 -1      # orientation of Burgers vector
   -bs      1/6          # scalar coefficient of Burgers vector, e.g., 1/2 in: 1/2<110>
   -dnorm   1  1  1      # normal direction of the plane with dislocation loop
   -dc      0. 0. 0      # center of dislocation loop
   -dr      10           # radius of dislocaiton loop
   -lx     -1  1  0      # local x axis on dislocation plane 
   -ly     -1 -1  2      # local y axis on dislocation plane 
   -ndisl   3            # segments/vertices of a discretized dislocation loop
